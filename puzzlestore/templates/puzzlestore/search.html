values() — Список словарей

Возвращает только указанные поля в виде словарей. Используется для сериализации в JSON.

{% for pub in publishers_values %} {% endfor %}
IDНазваниеСайт
{{ pub.id }} {{ pub.name }} {{ pub.website|default:"-" }}

values_list(flat=True) — Плоский список

Если указано одно поле и flat=True, возвращается простой список значений.

{{ game_names_flat }}

values_list() — Список кортежей

Возвращает список кортежей. Удобно для передачи данных в JavaScript.

{% for item in game_prices_list %} {% endfor %}
НазваниеЦена
{{ item.0 }} {{ item.1 }} руб.

count() — Подсчёт количества

Выполняет SQL SELECT COUNT(*). Работает быстрее, чем len(queryset), так как не загружает объекты в память Python.

Игр в наличии на складе: {{ games_in_stock_count }}

exists() — Проверка наличия

Выполняет SQL SELECT 1 ... LIMIT 1. Работает быстрее, чем if queryset, так как останавливает поиск после первой найденной записи.

Есть ли игры с нулевым остатком: {% if has_out_of_stock_games %} Да, требуют пополнения {% else %} Нет, всё в наличии {% endif %}

update() — Массовое обновление

Обновляет все записи одним SQL-запросом. Не вызывает save() и сигналы.
